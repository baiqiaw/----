### 5.2 技术决策

1. **技术栈确定**：
   - 后端：Node.js 20 + TypeScript + Fastify + Prisma
   - 前端：React 18 + TypeScript + Vite
   - 桌面客户端：Electron 28 + React
   - 数据库：SQLite 3（开发环境/个人版）/ PostgreSQL 14+（企业版生产环境）

2. **转换引擎确定**：
   - LibreOffice（Office文档）
   - Poppler（PDF处理）
   - Sharp（图片处理）
   - Pandoc（Markdown处理）

3. **OCR引擎确定**：
   - PaddleOCR（个人版）
   - 百度OCR（企业版）

4. **AI服务确定**：
   - 百度文心（摘要生成）
