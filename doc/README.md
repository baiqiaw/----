## 文档版本管理

所有文档均采用版本号管理，格式为：V主版本号.次版本号

- **主版本号**：重大变更或架构调整
- **次版本号**：内容更新或修订

## 文档更新流程

1. 文档创建或更新后，需在文档末尾的"更新记录"中记录
2. 重要文档更新需经过评审
3. 保持文档与代码同步更新
